Life Cycle of a Fixed Income Security

Fixed Income Securities is complex from accounting perspective to get all the dates and coupon calculation correct. The sheer volume and the breadth of this instruments makes it tough to accruately handle all possible situations. For e.g., during the life of the security, an investment managemr will need to accruately predict the cash flows given the security country of issuance and business day convention.

Coupon Date Fields
Name Description Usage
Issue Date The date the security was originally issued Adjusted Issue Price
Accrual Date First day the security starts accruing Earning Calculation Begins
First Coupon Date Date the security pays the first coupon Used for coupon calculation
Last Coupon Date Date of the last normal length coupon period Used for coupon calculation
Maturity Date Date the security matures Repays all outstanding principal and interest
